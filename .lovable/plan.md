

## Plan: Убрать задержку открытия чата Venyoo

### Проблема
Скрипт Venyoo загружается только по клику, поэтому первый клик тратит ~1-2 секунды на загрузку скрипта + ещё 500мс `setTimeout` перед автокликом виджета.

### Решение
1. **Предзагрузка скрипта** — загружать скрипт Venyoo сразу при монтировании компонента (в `useEffect`), а не по клику
2. **Клик без задержки** — при нажатии кнопки "Чат" сразу искать и кликать элемент виджета (без `setTimeout`). Если виджет ещё не инициализирован, использовать `MutationObserver` для отслеживания появления кнопки виджета и автоклика
3. **Скрытие нашей кнопки** — после успешного открытия виджета скрывать кастомную кнопку "Чат"

### Файл: `src/components/VenyooChat.tsx`

- В `handleClick`: сразу искать `.venyoo-activator` и кликать; если не найден — поставить `MutationObserver`
- Убрать `setTimeout(500)`
